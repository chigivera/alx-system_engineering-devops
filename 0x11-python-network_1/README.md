# Python Network #1

This project contains Python scripts that demonstrate various network programming concepts using `urllib` and `requests` packages.

## Tasks

### 0. What's my status? #0
* Script that fetches https://alx-intranet.hbtn.io/status using urllib

### 1. Response header value #0
* Script that displays X-Request-Id header from a URL

### 2. POST an email #0
* Script that sends POST request with email parameter

### 3. Error code #0
* Script that handles HTTP errors

### 4. What's my status? #1

### 5. Response header value #1
